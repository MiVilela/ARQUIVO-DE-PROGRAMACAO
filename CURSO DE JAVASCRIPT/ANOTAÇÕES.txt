## Operadores 
|Operador | Descrição | Operadores Encurtados |
|---------| --------- | ------------------- |
|  +    |    Adição    |x += y  |
|    -  |   Subtração | x -= y  |
|   /   |   Divisão   | x /= y  |
|   *   |   Multiplicação | x *= y |
| %     |   Módulo / Resto | x %= y|
| --    |   Decrementar | x --= y |
| ++    |   Incrementar | x ++= y |
| **    |   Exponencial | x **= y  |


## Operadores de comparação
|Operador | Descrição | 
|---------| --------- | 
| == Igual | Operandos sejam iguais retorna verdadeiro|
| != Não igual | Operandos não sejam iguais retorna verdadeiro|
| > Maior que | Operando da esquerda seja maior que o da direita retorna verdadeiro |
| => Maior ou igual que | Operando da esquerda seja maior ou igual ao da direita retorna verdadeiro |
| < Menor que | Operando da esquerda seja menor que o da direita retorna verdadeiro |
| = < Menor ou igual que | Operando da esquerda seja menor ou igual que o da direita retorna verdadeiro |


## DOM
Ele conecta páginas da web a scripts ou linguagens de programação, representando a estrutura de um documento.

Representa o documento como uma arvore lógica.

Arvore hierarquica, representada por parent e child 

Exemplo mais simples de todos para adicionar elementos 

<script/>

        window.document.write('Olá Mundo')

</script>


window.document.write(window.document.URL) 

Posso colocar na pagina a URL que estou acessando

Elemento é qualquer coisa que aparece na arvore DOM 

Posso selecionar esses elementos para navegar dentro da arvore

5 metodos de selecionar 

por Marca "getElementsByTagName()" - Posso selecionar mais de um elemento

por ID  

por Nome 

por Classe 

por Seletor
